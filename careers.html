Django Backend Developer Intern

Are you a Backend Developer who is skilled at rapid web development and specialize in using the Django framework?

Our organization is looking for a Django Developer who is extremely knowledgeable and a driven individual to assist the organization in designing and programming web applications. You will look after developing backend code which efficiently and effectively interfaces with front-end code to transact data. You will need to liaise with front-end developers extremely regularly, so being a good communicator and collaborator is extremely important.

Job Responsibilities:
  • As a Django Developer at our company, you will work with stakeholders and fellow developers.
  • Design and implement Python code using the Django framework.
  • Identify and fix bottlenecks that may arise from inefficient code.
  • Identify and fix software bugs.
  • Create a wide variety of unit tests to verify the functionality of the software.
  • Manage the security of the platform.
  • Write detailed documentation around the code.
  • Knowledge of front-end languages.
Job Requirements:
  • The ability to problem-solve and critically think.
  • High level of knowledge of Python and the Django framework.
  • Familiarity with event-driven programming as well as the MVC.
  • Good understanding of SQL databases.
  • Good understanding of REST APIs.
  • Knowledge of deployment on Linux servers
  • Knowledge of AWS services such as S3, EC2, and RDBMS
  • Knowledge of git

For qualification please create a Django Project that must include:
  1. django-allauth for user registration and login with basic UI
  2. CRUD operation on any model of your choice with class-based as well as function-based views with basic UI
  3. Test cases for CRUD operation using coverage package
  4. Upload code on GitHub private repository along with resume and share it with GitHub id: avdhootmukhedkar@gmail.com or sonalmukhedkar@gmail.com
  5. Host the project using Gunicorn and Ngninx and expose it using Ngrok during the interview

Interested candidates to attempt the above test for screening purposes. Even if you are not able to complete all the modules please submit a partially completed test project for code review which will help in assessing your technical skills.

Benefits:
  • Opportunity to work on live project.
  • Flexible timing and task-oriented work schedule
  • 100 % work from home
  • Exposure to full software development life cycle
  • Certificate on completion of 3 months internship
  • Stipend on a performance basis

Django Frontend Developer Intern

Are you a Frontend Developer who is skilled at rapid web development and specialize in using the Django framework?

Our organization is looking for a Django Developer who is extremely knowledgeable and a driven individual to assist the organization in designing and programming web applications. You will look after developing frontend code which efficiently and effectively interfaces with backend code to transact data. You will need to liaise with backend developers extremely regularly, so being a good communicator and collaborator is extremely important.

Job Responsibilities:
  • As a Django Developer at our company, you will work with stakeholders and fellow developers.
  • Directly contribute to the engineering team by pushing quality, tested code that other members of the team will be delighted to work with.
  • Design and implement user experiences to solve problems brought to their attention by members of the team.
  • See a problem – fix a problem. The Django Developer will be living in the codebases that our users directly interact with. If they come across a problem or pinpoint areas for improvement it is their responsibility to suggest and implement solutions.
  • Consistently implement web accessibility best practices. Accessibility is not optional; all work must meet accessibility standards.
  • Write detailed documentation around the code.
  • Knowledge of front-end languages.
Job Requirements:
  • High level of knowledge of Python and the Django framework.
  • Familiarity with event-driven programming as well as the MVC.
  • An excellent understanding of HTML,CSS,Javascript & Jquery
  • Not all HTML is good – candidates must understand how to create meaningful, semantic, accessible HTML.
  • Knowledge of CSS preprocessors like sass or less.
  • Experience with frameworks like Bootstrap, or Foundation.
  • Knowledge of Javascript Frameworks such as React
  • Solid understanding of Vanilla JS (core concepts) and working within frameworks.
  • Knowledge of git

For qualification please create a Django Project that must include:
  1. Base template with bootstrap5
  2. User registration screen extending base template
  3. CRUD operation on any model of your choice with class-based as well as function-based views which extends base UI
  4. Create components using django-classy-tags for breadcrumbs and pagination
  5. Test the application using selenium
  6. Upload code on GitHub private repository along with resume and share it with GitHub id: avdhootmukhedkar@gmail.com or sonalmukhedkar@gmail.com
  7. Host the project using Gunicorn and Ngninx and expose it using Ngrok during the interview

Interested candidates to attempt the above test for screening purposes. Even if you are not able to complete all the modules please submit a partially completed test project for code review which will help in assessing your technical skills.

Benefits:
  • Opportunity to work on live project.
  • Flexible timing and task-oriented work schedule
  • 100 % work from home
  • Exposure to full software development life cycle
  • Certificate on completion of 3 months internship
  • Stipend on a performance basis
